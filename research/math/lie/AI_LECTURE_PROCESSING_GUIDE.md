# AI处理讲义流程规范

作为AI处理讲义的标准化流程，请严格按照以下步骤执行：

## 1. 阅读讲义
仔细阅读并理解提供的讲义原文，掌握核心概念、整体脉络以及各个定理、引理和例子之间的逻辑关系。

## 2. 提取并编写 LaTeX 文档
将讲义内容转化为结构化的中文 LaTeX 文档。具体要求如下：
* **语言**：必须使用中文（支持中文的LaTeX环境，如 `ctex` 或 `CJKutf8`）。
* **专业术语解释**：在文档的开头部分，集中罗列并解释讲义中出现的所有重要专业术语。
* **按顺序陈述**：严格按照讲义的逻辑顺序，对讲义中的定理（Theorem）、引理（Lemma）、例子（Example）进行编号并清晰陈述。
* **证明与例子处理**：
    * **略去不重要的证明**：对于常规或不重要的证明，不需要进行整理和翻译。
    * **保留关键证明与例子**：对于非常重要的例子和核心证明，必须将其紧附在相应的定理或引理之后进行详细说明。

## 3. 编译与清理
* 使用适当的 LaTeX 引擎（如 `pdflatex` 或 `xelatex`）编译生成 PDF 文件。
* 编译成功后，务必清理并删除所有自动生成的编译中间文件（如 `.aux`, `.log`, `.out`, `.toc`, `.fls`, `.synctex.gz` 等），保持工作目录整洁。
